Little Lemon Capstone API
Available Endpoints:
Menu:
GET /api/menu/
POST /api/menu/
GET /api/menu/<id>/
PUT /api/menu/<id>/
DELETE /api/menu/<id>/
Bookings (requires Token):
GET /api/bookings/
POST /api/bookings/
GET /api/bookings/<id>/
PUT /api/bookings/<id>/
DELETE /api/bookings/<id>/
User Registration:
POST /api/registration/
Auth Tokens:
POST /api/token/
POST /api/token/refresh/
